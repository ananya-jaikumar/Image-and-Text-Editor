# 📝 Editor Application

## 📌 Project Overview
The **Editor Application** is a user-interactive project that provides two options for editing:
1. **Image Editor** 📷 - Edit a single image or batch process multiple images.
2. **Text Editor** 📝 - Create, modify, format, and manage text efficiently.

This project was developed as part of **CS 630005: Operating System Design** and focuses on leveraging OS functionalities for seamless editing capabilities.

## 🚀 Features
### 🔹 Image Editor
- **Single Image Editing** (Crop, Resize, Apply Filters)
- **Batch Processing** for multiple images
- **Text Extraction from Images** (OCR-powered)
- **Support for JPEG, PNG, GIF formats**

### 🔹 Text Editor
- **Create, Edit, and Save Text Files** (TXT, RTF, DOCX)
- **Advanced Formatting Options** (Font, Size, Color, Alignment)
- **Search & Replace Functionality** 🔍
- **Unicode & Multi-language Support** 🌍

## 🏗️ Tech Stack
- **Python** 🐍 (Core Logic)
- **Tkinter** 🎨 (GUI Framework)
- **Pillow (PIL)** 🖼️ (Image Processing)
- **Tesseract OCR** 🔠 (Text Extraction)

## 🛠️ Installation Guide
### 🔹 Prerequisites
Ensure you have the following installed:
- **Python 3.x**
- **Tkinter** (`pip install tk`)
- **Pillow (PIL)** (`pip install pillow`)
- **Tesseract OCR** (`pip install pytesseract`)

### 🔹 Running the Application
```bash
# Clone the repository
git clone https://github.com/your-repo/editor-app.git
cd editor-app

# Run the application
python editor.py
```
